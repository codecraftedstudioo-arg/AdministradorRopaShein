# @shein/utils

Utilidades **puras** (sin dependencias de framework) reutilizables por web,
mobile y store.

## Estructura prevista

```
src/
├── formatters/     # moneda, números, fechas legibles
├── validators/     # validaciones de dominio
├── date/           # utilidades de fecha
├── currency/       # centavos <-> unidades, formato
├── strings/        # slugify, iniciales, truncado
├── ids/            # generación de códigos/IDs
├── helpers/        # varios
└── index.ts
```

> En esta etapa solo se prepara la estructura.
