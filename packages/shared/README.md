# @shein/shared

Constantes y reglas transversales del dominio, compartidas por todo el
ecosistema.

## Estructura prevista

```
src/
├── constants/      # valores fijos (categorías base, límites, etc.)
├── enums/          # enums de aplicación
├── permissions/    # matriz de permisos por rol (ADMIN / EMPLOYEE)
├── config/         # configuración global de dominio
└── index.ts
```

> En esta etapa solo se prepara la estructura.
